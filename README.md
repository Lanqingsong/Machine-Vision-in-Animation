# 🤖 机器视觉动画演示教学平台 (Machine-Vision-in-Animation)

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![PySide6](https://img.shields.io/badge/GUI-PySide6-green.svg)](https://doc.qt.io/qtforpython/)
[![License](https://img.shields.io/badge/License-MIT-orange.svg)](LICENSE)

这是一个致力于通过**交互式动画**直观展示机器视觉底层原理的开源项目。相比于枯燥的公式和静态图片，本项目通过 Python 实时模拟成像过程、噪声模型、快门机制等，帮助学生真正理解机器视觉的底层逻辑。

---

## 🚀 软件下载与安装 (推荐学生使用)

为了方便学生使用，本项目已打包成绿色版 `.exe` 软件。**无需安装 Python 环境，解压即用。**

  链接：https://pan.baidu.com/s/1DpoqbZNErwVDULh5ufobxw](https://pan.baidu.com/s/1wva-sX_QwYadqp1sTa_Sdw 提取码：ufvx 
### 2. 解压后的目录结构

下载压缩包解压后，目录结构如下：

```
📁 机器视觉教学平台/
├── 机器视觉教学平台.exe      ← 主程序，双击启动
├── python/                   ← 内置便携 Python 环境（请勿删除）
│   ├── python.exe
│   └── Lib/site-packages/    ← numpy、matplotlib、pillow、opencv 等依赖
└── lessons/                  ← 课程脚本存放目录（同步后自动生成）
    ├── CH-1.py
    ├── CH-2.py
    └── ...
```

> ⚠️ **注意**：请将整个文件夹解压到一个**固定路径**（如 `D:\CV_COURSE\`），不要放在桌面临时目录或带中文/空格的路径下。所有文件夹必须保持相对位置不变。

### 3. 使用方法

**首次使用：**

1. 解压压缩包到固定文件夹
2. 双击运行 `机器视觉教学平台.exe`
3. 观察左上角 Python 状态标签：
   - 🟢 **`✔ 便携 Python`** — 环境正常，可以使用
   - 🔴 **`✘ 未找到 Python`** — 文件夹结构被破坏，请重新解压
4. 点击 **【☁ 同步/更新课程】**，程序自动从 GitHub 下载最新课程脚本到 `lessons/` 目录
5. 在列表中选择课程，点击 **【▶ 启动选中课程】** 即可观看交互演示

**日常使用：**

- 直接双击 `机器视觉教学平台.exe`，选课启动即可
- 有新课发布时，点击【☁ 同步/更新课程】获取最新内容，无需重新下载整个软件

## 📚 课程内容概览

### 第一章：传感器基础

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-1 | 传感器参数仪表盘 | 调节分辨率、像素尺寸、帧率，实时观察传感器物理尺寸与数据带宽变化 |
| CH-2 | 采样定理与分辨率 | 直观感受 FOV、像素精度与最小缺陷检测能力的关系 |

### 第三章：快门机制

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-3_0 | 快门机制对比（几何版） | 动画演示全局快门同步曝光 vs 滚动快门逐行扫描的畸变原理 |
| CH-3_1 | 快门机制对比（图像版） | 带真实图像与曝光时序图的完整对比演示 |

### 第四章：传感器噪声

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-4 | 噪声模型与 SNR | 调节光照、曝光时间、读出噪声、暗电流，观察信噪比的非线性变化与噪声来源分解 |

### 第五章：动态范围与 HDR

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-5 | 动态范围限制与 HDR 效果 | 对比受限动态范围与 HDR 处理后的成像差异 |
| CH-5_1 | 动态范围滑块演示 | 调节动态范围档数与曝光偏移，高亮过曝/欠曝区域 |
| CH-5_2 | 多曝光 HDR 融合 | 演示欠曝、正常、过曝三张图像融合为 HDR 图像的过程 |
| CH-5_3 | 色调映射算法对比 | 交互对比 Linear / Gamma / Reinhard 三种 HDR 色调映射算法 |

### 第六章：接口选型

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-6 | 相机接口选型指南 | 通过距离与带宽滑块，交互推荐 USB 3.0 / GigE / Camera Link / CoaXPress 接口 |

### 第七章：镜头光学基础

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-7 | 焦距计算光路图 | 基于相似三角形原理，交互演示 f = WD × S / FOV 的计算过程 |
| CH-7_1 | 镜头参数联动光路图 | 焦距、工作距离、传感器尺寸三参数动态联动，实时更新光路与 FOV |
| CH-7_2 | 景深与光圈模糊模拟 | 调节 F-stop，观察前景/中景/背景的焦外模糊程度与画面亮度联动 |
| CH-7_3 | 弥散圆（CoC）演示 | 调节光圈大小与非对焦物距，实时计算并可视化弥散圆直径 |

### 第八章：景深计算

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-8 | 景深计算仪表盘 | 焦距、光圈、对焦距离三参数联动，双视图展示光锥与景深范围 |

### 第九章：远心镜头

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-9 | 普通镜头 vs 远心镜头 | 拖动物距滑块，对比两种镜头在放大率随距离变化上的本质差异 |

### 第十章：法兰距与透视畸变

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-10 | C/CS-Mount 法兰距匹配 | 交互选择镜头与相机型号及适配环，判断是否能清晰成像 |
| CH-10_1 | 普通 vs 远心镜头深孔成像 | 2×2 布局展示深孔/圆柱体在两种镜头下的光路与成像差异 |

### 第十一章：镜头畸变

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-11 | 球差原理演示 | 调节球差系数，对比理想近轴光线与实际边缘光线的偏差及放大率变化 |
| CH-11_1 | 径向畸变棋盘格演示 | 调节 k1/k2，观察桶形/枕形畸变对棋盘格的变形影响及测量误差 |
| CH-11_2 | 畸变校正演示 | 在已知畸变图像上调节校正系数，模拟迭代逆映射校正过程 |

### 第十二章：像素当量

| 脚本 | 主题 | 说明 |
|------|------|------|
| CH-12 | 像素当量计算 | 交互演示 FOV 与分辨率如何共同决定每像素对应的物理尺寸（mm/pixel） |

## 🛠 开发者环境配置 (运行源代码)

如果你希望直接运行或修改源代码，请参考以下步骤：

### 1. 创建 Conda 环境

```bash
conda create -n mv_env python=3.10 -y
conda activate mv_env
```

### 2. 安装依赖库

```bash
pip install PySide6 requests matplotlib numpy pillow opencv-python
```

### 3. 运行平台主程序

```bash
python platform_launcher.py
```

### 4. 直接运行单个课程脚本

```bash
python py/CH-1.py
```
